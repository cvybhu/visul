# visul
Music Visualizator / projector lights system

Records sound via pulseaudio API and creates visualization

make debug/release
to build


[USED LIBRARIES]:

PULSEAUDIO:
sudo apt install libpulse-dev

SFML:
sudo apt install libsfml-dev

FFTW 3.3.8:
http://www.fftw.org/download.html
./configure
make
sudo make install
